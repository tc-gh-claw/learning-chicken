# 🐔 學習雞 - Learning Chicken

> 一款結合學習與養成的趣味教育遊戲

## 🎮 遊戲簡介

「學習雞」是一款專為小學生設計的養成遊戲。玩家通過回答不同科目的題目獲得飼料，餵食小雞幫助它成長，最終進化成獨特的形態！

### 核心循環

```
選擇科目答題 → 獲得對應飼料 → 餵食學習雞 → 累計答對20題 → 進化！
```

## ✨ 遊戲特色

### 📚 四個學科目
- 📝 **中文** - 語文知識、成語、詩詞
- 🔤 **英文** - 單詞、語法、句型
- 🔢 **數學** - 計算、幾何、應用題
- 🌍 **常識** - 科學、地理、生活知識

### 🎯 年級分級
- 題目難度覆蓋 **小一至小六**
- 共 **120題** 精心設計題目
- 循序漸進，寓教於樂

### 🐔 30種進化形態

根據答題表現，學習雞會進化成不同的形態：

| 類型 | 數量 | 例子 |
|------|------|------|
| 🎓 全能型 | 5種 | 全能學霸雞、知識博學雞 |
| 🔢 數學型 | 4種 | 數學天才雞、計算大師雞 |
| 📝 語文型 | 4種 | 語文大師雞、文學才子雞 |
| 🔤 英文型 | 4種 | 英文流利雞、外語達人雞 |
| 🌍 常識型 | 4種 | 常識達人雞、科學探索雞 |
| 🎭 彩蛋型 | 9種 | 偏科冒險雞、臨時抱佛腳雞 |

## 🚀 開始遊戲

### 本地運行

```bash
# 克隆項目
git clone https://github.com/tc-gh-claw/learning-chicken.git
cd learning-chicken

# 用瀏覽器打開
open index.html
```

### 在線遊玩

👉 **[點擊這裡開始遊戲](https://tc-gh-claw.github.io/learning-chicken/)**
